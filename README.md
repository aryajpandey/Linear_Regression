# Linear Regresion
This project is an example of a lLinear Regression model using scikit-learn and is written in Python 3.8.

	Libraries used :
		numpy
		pandas
		scikit-learn
		matplotlib
		seaborn

Any of these libraries can be installed via pip or condo command in any Operating System.
